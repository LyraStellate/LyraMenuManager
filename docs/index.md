features:
  - title: 🎮 VRChat完全再現のUI
    details: VRChatゲーム内と完全に同じレイアウトの円形（ラジアル）メニューUIをエディター上にそのまま再現。完成イメージを直接見ながら編集が可能です。
  - title: 🖱️ 直感的なドラッグ＆ドロップ操作
    details: アイテムを掴んでドラッグし、順番の入れ替えやサブメニューへの移動など、直感的に階層を組み替えることができます。
  - title: 🔧 Modular Avatar 完全対応
    details: アバター内に分散している ModularAvatarMenuInstaller を自動で集約・リストアップ。散らばったメニュー設定を探し回る必要がありません。
  - title: 📦 便利なインベントリシステム
    details: 今は使わないアイテムを画面左側の「インベントリ」に一時退避。後から自由に整理し直すことができます。
  - title: 🛡️ 完全非破壊での適用
    details: 変更はNDMFのビルド時に自動適用。元のMAコンポーネントやアセットを書き換えない安全設計です。
  - title: 📂 超過メニューの自動処理
    details: 1階層8アイテムを超過した場合、VRChat公式と同じ仕様で自動的にサブメニューフォルダを生成して格納します。
